X-Port

A userscript to help export favorites on E*Hentai.
Just go to your favorites page and click the "Export..." button next to the search field, then wait for it to complete.

Install

First, install any one of the userscript managers listed for your web browser below, then click the install link to get the userscript.

Firefox

  • Violentmonkey
  • Tampermonkey
  • Greasemonkey 4+ (For Firefox versions 57+)
  • Greasemonkey 3.17 (For versions earlier than 57)

Chrome

  • Tampermonkey
